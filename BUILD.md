# 构建说明

## 当前构建

- 版本：1.0.1
- 构建时间：2026-04-19
- 构建方式：GitHub Actions + EAS Build
- 构建配置：preview profile（支持完整功能）

## 功能说明

### ✅ 已实现功能

1. **复习提醒通知**
   - 支持设置每日复习提醒时间（00:00-23:59）
   - 支持开启/关闭提醒
   - 通知功能仅在开发构建或生产构建中可用
   - **注意**：Expo Go 环境不支持通知功能

2. **FSRS 智能复习算法**
   - 自适应间隔复习
   - 根据记忆表现调整复习周期

3. **词书管理**
   - 支持自定义词书
   - 支持导入/导出单词
   - 支持多种导入格式

## 技术栈

- 前端：Expo 54 + React Native
- 后端：Express.js
- 数据库：SQLite (expo-sqlite)
- 通知：expo-notifications

## 安装说明

1. 从 GitHub Actions 下载 APK
2. 在 Android 设备上安装
3. 首次使用时授予通知权限
4. 在"复习提醒"页面设置提醒时间

## 已知限制

- iOS 平台暂不支持（仅 Android）
- Web 预览不支持通知功能
- Expo Go 不支持通知功能
