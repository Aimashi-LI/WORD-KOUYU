## 一、Android 权限说明

### 1.1 存储权限（可选）

**权限名称**：`android.permission.READ_EXTERNAL_STORAGE` / `android.permission.WRITE_EXTERNAL_STORAGE`

**用途说明**：
- 导出词库数据为 CSV/JSON 文件到设备存储
- 从设备存储导入词库文件

**请求时机**：
- 仅在您点击"导出词库"或"导入词库"功能时请求
- 您可以拒绝授权，不影响其他功能使用

**数据访问范围**：
- ✅ 应用私有目录：自动访问，无需权限
- ✅ 导出/导入的文件：仅访问您明确选择的文件
- ❌ 其他文件：不会访问您的其他文件

**是否必需**：❌ **否**（可选权限）

## 三、权限请求时机

本应用遵循"按需请求"原则，不会在应用启动时批量请求权限。

### 权限请求流程

1. **应用启动**：不请求任何权限
2. **使用功能时**：仅在使用需要权限的功能时请求
3. **拒绝后处理**：您可以拒绝授权，应用会提示您该功能受限，但不会强制要求授权

### 示例

```
场景：用户点击"导出词库"按钮

1. 系统弹窗：请求存储权限
   选项：[允许] [仅在使用时允许] [拒绝]

2. 如果用户选择"允许"：
   → 继续导出流程

3. 如果用户选择"拒绝"：
   → 提示"需要存储权限才能导出文件"
   → 返回上一页（不强制要求授权）
```

## 四、权限管理

### Android 权限管理

您可以在系统设置中管理应用权限：

1. 打开**设置** → **应用** → **编码记忆法**
2. 点击**权限**
3. 对每个权限进行授权或撤销

### iOS 权限管理

您可以在系统设置中管理应用权限：

1. 打开**设置** → **编码记忆法**
2. 对每个权限开关进行开启或关闭

## 七、常见问题

### Q1: 为什么应用需要存储权限？
**A**: 仅在您导出词库到设备存储或从设备导入词库时需要。您可以选择不授权，改用应用内浏览功能。

### Q2: 应用会读取我的照片吗？
**A**: 不会。照片库权限仅在您选择导出文件保存位置或选择要导入的文件时使用，不会读取您的其他照片。

### Q3: 应用需要网络权限吗？
**A**: 不需要。本应用完全离线运行，不请求网络权限。

### Q4: 为什么有些权限在应用商店显示"可能使用"？
**A**: 这是因为应用使用了第三方 SDK（如文档选择器），这些 SDK 可能支持某些功能，但本应用不使用这些功能。

### Q5: 我可以撤销已授权的权限吗？
**A**: 可以。您可以在系统设置中随时撤销任何已授权的权限。撤销后，相关功能将受限，但不影响应用其他功能使用。

### Q6: 应用会在后台使用相机或麦克风吗？
**A**: 不会。本应用不请求相机和麦克风权限，也无法在后台使用这些功能。

### Q7: 权限数据会被上传吗？
**A**: 不会。本应用完全离线运行，不会上传任何数据到服务器。
