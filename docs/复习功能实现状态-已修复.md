## 一、核心算法功能

### 1. FSRS 动态间隔算法 ✓ 已实现
**位置**: `client/algorithm/fsrs.ts`

**功能**:
- `calculateRetrievability()` - 计算可提取性 R 值
  - 公式: R = exp(-间隔天数 / S)
  - 用于判断单词是否需要复习
- `predictNextInterval()` - 预测下次复习间隔（天）
  - 基于得分和当前稳定性计算
  - 回答正确时稳定性增加，回答错误时稳定性降低
- `needsReview()` - 检查单词是否需要复习
  - R < 0.9 时加入待复习队列

**实现状态**: ✓ 已正确实现

### 2. 回忆时间加权机制 ✓ 已实现
**位置**: `client/algorithm/fsrs.ts`

**功能**:
- `calculateTimeBudget()` - 计算时间预算
  - 基础时间 + 难度加成
- `updateWithTimeWeight()` - 回忆时间加权更新
  - 根据回忆时间调整得分权重
  - 响应快 → 提权（延长间隔）
  - 响应慢 → 降权（缩短间隔）
- `calculateWeightedScore()` - 计算加权得分
  - 快速回答：weight = 1.2
  - 正常回答：weight = 1.0
  - 慢速回答：weight = 0.8

**实现状态**: ✓ 已正确实现

**改进效果**:
- 考虑字符顺序（编辑距离）
- 考虑字符组成（字符集合）
- 考虑字符顺序但不要求连续（最长公共子序列）
- 加权平均，编辑距离权重最高（50%）
