# 快速开始 - 5分钟打包 APK

## 🚀 最简步骤（Android）

### 第一步：准备环境（5分钟）

1. **安装 Node.js**
   - 访问：https://nodejs.org/
   - 下载并安装 LTS 版本

2. **安装 Java JDK**
   - 访问：https://adoptium.net/
   - 下载并安装 Temurin JDK 17

3. **安装 Android Studio**
   - 访问：https://developer.android.com/studio
   - 下载并安装，完成初始设置向导

### 第二步：配置环境变量（2分钟）

**Windows:**
```cmd
# 设置环境变量（在"系统属性"中设置）
JAVA_HOME = C:\Program Files\Eclipse Adoptium\jdk-17.0.9.101-hotspot
ANDROID_HOME = C:\Users\你的用户名\AppData\Local\Android\Sdk

# 添加到 Path
%JAVA_HOME%\bin
%ANDROID_HOME%\platform-tools
%ANDROID_HOME%\tools
```

**macOS / Linux:**
```bash
# 添加到 ~/.zshrc
echo 'export JAVA_HOME=$(/usr/libexec/java_home -v 17)' >> ~/.zshrc
echo 'export ANDROID_HOME=$HOME/Android/Sdk' >> ~/.zshrc
echo 'export PATH=$PATH:$ANDROID_HOME/platform-tools' >> ~/.zshrc
source ~/.zshrc
```

### 第三步：获取项目并安装依赖（3分钟）

```bash
# 1. 解压项目到工作目录
cd ~/Projects/word-review/client

# 2. 安装依赖
npm install
```

### 第四步：打包 APK（10-15分钟）

```bash
# 1. 生成原生代码
npx expo prebuild --clean

# 2. 进入 Android 目录
cd android

# 3. 构建 Debug APK
./gradlew assembleDebug

# 4. APK 文件位置
ls app/build/outputs/apk/debug/app-debug.apk
```

### 第五步：安装到手机（2分钟）

**方法 1：通过 ADB**
```bash
# 连接手机，启用 USB 调试
adb install app/build/outputs/apk/debug/app-debug.apk
```

**方法 2：直接传输**
1. 通过 USB/蓝牙将 APK 传输到手机
2. 点击 APK 文件，允许安装
3. 完成安装

## ❓ 常见问题

### Q1：`adb` 命令找不到？
```bash
# 检查 ANDROID_HOME 环境变量
echo $ANDROID_HOME  # macOS/Linux
echo %ANDROID_HOME%  # Windows

# 确保 ANDROID_HOME/platform-tools 在 Path 中
```

### Q2：Gradle 构建失败？
```bash
# 清理并重新构建
cd android
./gradlew clean
./gradlew assembleDebug
```

### Q3：APK 安装失败？
```bash
# 卸载旧版本
adb uninstall com.wordreview.app

# 重新安装
adb install app/build/outputs/apk/debug/app-debug.apk
```
