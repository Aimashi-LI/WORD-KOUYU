## 🔧 技术选型

### 为什么选择 Vite？

- ⚡ 快速的热更新
- 📦 开箱即用的 TypeScript 支持
- 🎨 无需配置即可使用
- 🚀 构建速度快

### 为什么选择 localStorage？

- 💾 简单易用
- 🔒 数据本地存储
- 🚫 无需后端
- 📱 支持离线使用

### 为什么保持 React？

- 🔄 与移动版技术栈一致
- 📚 生态丰富
- 🛠️ 开发效率高
