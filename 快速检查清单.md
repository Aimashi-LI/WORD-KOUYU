# Windows + Android 打包 - 快速检查清单

## ✅ 准备阶段

### 1. 安装软件
- [ ] 下载并安装 Node.js（LTS 版本）
- [ ] 下载并安装 Java JDK 17（Temurin）
- [ ] 下载并安装 Android Studio
- [ ] 启动 Android Studio，完成初始设置向导

### 2. 验证安装
- [ ] 打开 CMD，输入 `node --version`（显示版本号）
- [ ] 输入 `java -version`（显示版本信息）
- [ ] 打开 Android Studio → Tools → SDK Manager
- [ ] 确认已安装 Android 14.0 (API 34)

---

## ⚙️ 配置阶段

### 3. 配置环境变量
- [ ] 按 `Win + R`，输入 `sysdm.cpl`，打开环境变量
- [ ] 新建 `JAVA_HOME` = `C:\Program Files\Eclipse Adoptium\jdk-17.0.9.101-hotspot`
- [ ] 新建 `ANDROID_HOME` = `C:\Users\你的用户名\AppData\Local\Android\Sdk`
- [ ] 编辑 `Path`，添加：
  - [ ] `%JAVA_HOME%\bin`
  - [ ] `%ANDROID_HOME%\platform-tools`
  - [ ] `%ANDROID_HOME%\tools`

### 4. 验证环境变量
- [ ] 关闭所有 CMD 窗口，重新打开
- [ ] 输入 `echo %JAVA_HOME%`（显示路径）
- [ ] 输入 `echo %ANDROID_HOME%`（显示路径）
- [ ] 输入 `adb version`（显示版本信息）

---

## 📦 项目阶段

### 5. 准备项目
- [ ] 创建 `word-review` 文件夹
- [ ] 将项目解压到该文件夹
- [ ] 进入 `client` 目录
- [ ] 确认能看到 `package.json` 文件

### 6. 安装依赖
- [ ] 打开 CMD，进入 `client` 目录
- [ ] 输入 `npm install`
- [ ] 等待安装完成（3-10 分钟）
- [ ] 确认生成了 `node_modules` 文件夹

---

## 🔨 打包阶段

### 7. 生成原生代码
- [ ] 在 `client` 目录下，输入 `npx expo prebuild --clean`
- [ ] 等待完成（2-5 分钟）
- [ ] 确认生成了 `android` 文件夹

### 8. 构建 APK
- [ ] 进入 `android` 目录
- [ ] 输入 `gradlew.bat assembleDebug`
- [ ] 等待完成（15-30 分钟）
- [ ] 看到 `BUILD SUCCESSFUL` 提示

### 9. 查找 APK
- [ ] 确认文件位置：`android\app\build\outputs\apk\debug\app-debug.apk`
- [ ] 在文件资源管理器中打开该文件夹
- [ ] 确认 APK 文件存在（至少 10MB）

---

## 📲 安装阶段

### 10. 准备手机
- [ ] 打开手机设置 → 关于手机
- [ ] 连续点击"版本号" 7 次，开启开发者模式
- [ ] 打开"开发者选项" → 开启"USB 调试"
- [ ] 打开"安全" → 允许"安装未知来源应用"

### 11. 安装 APK

#### 方法 A：通过 ADB（推荐）
- [ ] 用 USB 线连接手机到电脑
- [ ] 手机上允许 USB 调试
- [ ] 在 CMD 中输入 `adb devices`（看到设备列表）
- [ ] 输入 `adb install app\build\outputs\apk\debug\app-debug.apk`
- [ ] 看到 `Success` 提示

#### 方法 B：通过文件传输
- [ ] 用 USB 线连接手机到电脑
- [ ] 将 APK 文件复制到手机
- [ ] 在手机上点击 APK 文件
- [ ] 允许安装未知来源应用
- [ ] 点击"安装"

### 12. 启动应用
- [ ] 在手机上找到"编码记忆法"应用图标
- [ ] 点击打开
- [ ] 等待数据库初始化
- [ ] 开始使用！

## 🎯 核心命令速查

```cmd
# 验证环境
node --version
java -version
adb version

# 进入项目目录
cd C:\Users\你的用户名\Desktop\word-review\client

# 安装依赖
npm install

# 生成原生代码
npx expo prebuild --clean

# 构建 APK
cd android
gradlew.bat assembleDebug

# 安装到手机
adb install app\build\outputs\apk\debug\app-debug.apk

# 查看手机日志（用于调试）
adb logcat
```

## ❓ 快速问题解决

| 问题 | 解决方法 |
|------|---------|
| `node` 命令找不到 | 检查环境变量 Path |
| `java` 命令找不到 | 检查 JAVA_HOME 和 Path |
| `adb` 命令找不到 | 检查 ANDROID_HOME 和 Path |
| `npm install` 失败 | 使用国内镜像，重新安装 |
| Gradle 构建失败 | `gradlew.bat clean`，重新构建 |
| 手机连接不上 | 重启 ADB，更换 USB 线 |
| APK 安装失败 | `adb uninstall com.wordreview.app`，重新安装 |

---

## 📝 重要提示

1. **必须关闭所有 CMD 窗口后重新打开**，环境变量才能生效
2. **首次构建 APK 会很慢**（15-30 分钟），耐心等待
3. **必须开启 USB 调试**，否则无法连接手机
4. **必须允许安装未知来源应用**，否则无法安装 APK
5. **完全单机运行**，安装后无需联网
