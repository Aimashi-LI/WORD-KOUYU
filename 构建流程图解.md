# 📸 构建流程图解（一看就懂）

## 🎯 整体流程（只需 7 个步骤）

```
📥 下载项目 → 💻 安装依赖 → 🏗️ 生成 Android 项目 → 🎨 用 Android Studio 打开 → 📱 连接手机 → ▶️ 构建运行 → 🧪 测试识别
```

---

## 步骤详解

### 📥 第 1 步：下载项目代码

```
┌─────────────────────────────────┐
│  找到项目文件                    │
├─────────────────────────────────┤
│  📁 projects/                   │
│     ├─ 📁 client/    ← 重点是这里│
│     ├─ 📁 server/               │
│     └─ 📄 其他文件...            │
└─────────────────────────────────┘
           ↓
   复制到电脑 D 盘
```

**如果你看到这样的结构，就对了！**

---

### 💻 第 2 步：安装依赖

```
┌─────────────────────────────────┐
│  打开命令窗口                    │
├─────────────────────────────────┤
│  按 Win + R                     │
│  输入 cmd                       │
│  按回车                         │
└─────────────────────────────────┘
           ↓
┌─────────────────────────────────┐
│  进入项目文件夹                  │
├─────────────────────────────────┤
│  cd D:\workspace\projects\client│
│  按回车                         │
└─────────────────────────────────┘
           ↓
┌─────────────────────────────────┐
│  安装依赖                        │
├─────────────────────────────────┤
│  pnpm install                   │
│  按回车                         │
│  等待 2-5 分钟                  │
└─────────────────────────────────┘
           ↓
     看到 Done 就完成了！
```

---

### 🏗️ 第 3 步：生成 Android 项目

```
┌─────────────────────────────────┐
│  继续在命令窗口操作              │
├─────────────────────────────────┤
│  输入命令                        │
│                                 │
│  npx expo prebuild              │
│  --platform android --clean     │
└─────────────────────────────────┘
           ↓
     等待 2-5 分钟
           ↓
┌─────────────────────────────────┐
│  成功标志                        │
├─────────────────────────────────┤
│  client/ 文件夹里多了一个       │
│  📁 android/                    │
└─────────────────────────────────┘
```

---

### 🎨 第 4 步：用 Android Studio 打开

```
┌─────────────────────────────────┐
│  打开 Android Studio            │
├─────────────────────────────────┤
│  1. 双击图标                    │
│  2. 或者在开始菜单搜索          │
└─────────────────────────────────┘
           ↓
┌─────────────────────────────────┐
│  打开项目                        │
├─────────────────────────────────┤
│  File → Open...                 │
│  → 选择 client/android 文件夹   │
│  → 点击 OK                      │
└─────────────────────────────────┘
           ↓
┌─────────────────────────────────┐
│  等待 Gradle 同步                │
├─────────────────────────────────┤
│  底部显示进度条                  │
│  第一次需要 10-30 分钟          │
│  期间不要关闭                    │
└─────────────────────────────────┘
           ↓
    看到 BUILD SUCCESSFUL
```

---

### 📱 第 5 步：连接手机

```
┌─────────────────────────────────┐
│  手机设置（只需要做一次）        │
├─────────────────────────────────┤
│  1. 设置 → 关于手机             │
│  2. 连续点击"版本号" 7 次       │
│  3. 返回设置 → 开发者选项       │
│  4. 打开 USB 调试               │
└─────────────────────────────────┘
           ↓
┌─────────────────────────────────┐
│  连接电脑                        │
├─────────────────────────────────┤
│  1. 用 USB 线连接手机和电脑     │
│  2. 手机弹出授权窗口             │
│  3. 点击"允许"或"OK"            │
└─────────────────────────────────┘
           ↓
┌─────────────────────────────────┐
│  Android Studio 验证             │
├─────────────────────────────────┤
│  顶部工具栏能看到手机型号        │
│  比如：Xiaomi 14                │
└─────────────────────────────────┘
```

---

### ▶️ 第 6 步：构建并运行

```
┌─────────────────────────────────┐
│  在 Android Studio 操作          │
├─────────────────────────────────┤
│  1. 确认选择了你的手机           │
│  2. 点击绿色播放按钮 ▶️          │
│  3. 等待 3-10 分钟              │
└─────────────────────────────────┘
           ↓
┌─────────────────────────────────┐
│  应用自动安装到手机              │
├─────────────────────────────────┤
│  安装完成后自动打开              │
│  🎉 成功！                       │
└─────────────────────────────────┘
```

---

### 🧪 第 7 步：测试识别

```
┌─────────────────────────────────┐
│  打开应用                        │
├─────────────────────────────────┤
│  → 找到"拍照识别"               │
│  → 点击进入                      │
└─────────────────────────────────┘
           ↓
┌─────────────────────────────────┐
│  拍照识别                        │
├─────────────────────────────────┤
│  1. 对准英文单词                │
│  2. 放在扫描框内                │
│  3. 点击拍照                    │
│  4. 等待 1-2 秒                 │
└─────────────────────────────────┘
           ↓
┌─────────────────────────────────┐
│  查看结果                        │
├─────────────────────────────────┤
│  ✅ 成功识别单词！              │
│  ✅ 完全离线！                  │
└─────────────────────────────────┘
```

## 📊 时间预估

| 步骤 | 需要时间 | 说明 |
|------|---------|------|
| 下载项目 | 3 分钟 | 根据网络速度 |
| 安装依赖 | 5 分钟 | 首次下载 |
| 生成 Android 项目 | 5 分钟 | 生成原生代码 |
| Android Studio 同步 | 10-30 分钟 | 仅首次 |
| 连接手机 | 5 分钟 | 只需配置一次 |
| 构建运行 | 5-10 分钟 | 首次较慢 |
| 测试识别 | 3 分钟 | 验证功能 |
| **总计** | **30-60 分钟** | 首次构建 |

## 🎯 最重要的事情

1. **不要着急**：第一次构建慢是正常的
2. **保持网络**：构建过程需要联网下载依赖
3. **不要关闭**：构建期间不要关闭 Android Studio
4. **耐心等待**：Gradle 同步可能需要 10-30 分钟
